# packages/app/src/renderer/processes

## 目录用途

渲染层流程安装目录，负责把协议事件和协议请求接入状态层。

## 当前内容

| 目录                          | 说明                     |
| ----------------------------- | ------------------------ |
| `protocol-event-pipeline/`    | 通知事件入库与时间线写入 |
| `protocol-request-responder/` | 审批、用户输入等请求响应 |

## 维护边界

- ✅ 这里只负责编排入口与路由
- ❌ 不承担视图细节
