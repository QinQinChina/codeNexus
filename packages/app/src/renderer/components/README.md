# src/renderer/components

## 目录用途

渲染层组件目录，负责页面骨架、业务视图与通用 UI 组件。

## 当前结构

| 目录         | 说明                                       |
| ------------ | ------------------------------------------ |
| `layout/`    | 应用骨架、Settings、侧栏、聊天与编辑器布局 |
| `timeline/`  | 时间线卡片壳与卡片内容                     |
| `ui/`        | 通用交互组件                               |
| `workflow/`  | 工作流编辑、运行与画布                     |
| `guardian/`  | guardian 诊断组件                          |
| `mcp/`       | MCP 资源面板                               |
| `userInput/` | 用户输入挂载组件                           |

## 维护边界

- ✅ 组件层负责展示与交互编排
- ❌ 复杂业务规则优先下沉到 `features/` 或 `stores/`
